## Описание проекта
Нужно написать программу, которая угадывает загаданное компьютером число за минимальное количество попыток.

[к оглавлению](#оглавление)

---

## Какой кейс решаем
Требуется реализовать алгоритм угадывания числа и проверить, насколько эффективно он работает.

**Условия задачи:**
- компьютер загадывает целое число;
- программа должна его угадать;
- необходимо оценить, за сколько попыток в среднем алгоритм находит число.

**Метрика качества:**  
Среднее количество попыток за 1000 повторений.

**Что практикуем:**
- написание функций на Python;
- работу с циклами и условиями;
- использование библиотеки `numpy`;
- оценку эффективности алгоритма.

[к оглавлению](#оглавление)

## Краткая информация о данных
Данные генерируются автоматически:
- компьютер загадывает случайные числа;
- для проверки алгоритма используется 1000 повторений;
- генерация случайных чисел выполняется с помощью `numpy`.

[к оглавлению](#оглавление)

## Этапы работы над проектом
1. Создана функция угадывания числа.
2. Реализован подсчёт количества попыток.
3. Написана функция тестирования алгоритма на 1000 случайных числах.
4. Посчитано среднее количество попыток.
5. Проверена корректность работы программы.
